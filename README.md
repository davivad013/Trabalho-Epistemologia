## Insturcões para abrir o site

1. Faça um ambiente virtual usando o comando:
```bash 
    python -m venv venv
```
2. Ative o ambiente virtual usando o comando:
```bash 
    source venv/bin/activate
```
3. Instale as dependências usando o comando:
```bash 
    pip install -r requirements.txt
```
4. Vá para a pasta app:
```bash 
    cd app
```
5. Execute o aplicativo usando o comando:
```bash 
    python app.py
```
